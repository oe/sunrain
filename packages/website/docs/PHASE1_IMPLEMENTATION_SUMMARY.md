## 🚀 使用方法

### 访问练习功能
1. **从首页**: 点击 "Daily Practice" 或 "Quick Relax" 按钮
2. **直接访问**: 访问 `/{lang}/practice/` 页面
3. **选择练习**: 浏览练习列表，使用筛选器找到合适的练习
4. **开始练习**: 点击练习卡片进入练习播放器
5. **完成练习**: 按照步骤指导完成练习，记录心情变化

### 练习类型
- **呼吸练习**: 基础正念呼吸，适合初学者
- **冥想练习**: 身体扫描、慈悲冥想等
- **放松练习**: 渐进式肌肉放松等
